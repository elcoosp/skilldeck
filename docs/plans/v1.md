## 1. Project Overview & Architecture Summary

### 1.1 System Overview

SkillDeck v1 is a **local-first, reactive, event-driven state machine** wrapped in a Tauri desktop shell. The system is organized into three layers:

```
┌─────────────────────────────────────────────────────────────────┐
│                     REACT FRONTEND                               │
│  (Pure View Layer — communicates only via Tauri IPC)            │
│                                                                  │
│  Zustand (UI state)    TanStack Query (server state)            │
│  shadcn/ui components  React 19                                  │
└────────────────────────────┬────────────────────────────────────┘
                             │
                             │ Tauri IPC (invoke + events)
                             ▼
┌─────────────────────────────────────────────────────────────────┐
│                     TAURI SHELL                                  │
│  (Thin OS Integration Layer)                                     │
│                                                                  │
│  AppState registration   Command handlers                        │
│  Event bridging          OS keychain bridge                      │
│  Approval gate queue     Window management                       │
└────────────────────────────┬────────────────────────────────────┘
                             │
                             │ Direct API calls
                             ▼
┌─────────────────────────────────────────────────────────────────┐
│                     RUST CORE (skilldeck-core)                   │
│  (Pure Library Crate — Zero Tauri Dependency)                    │
│                                                                  │
│  Agent Layer      Skill Layer       MCP Layer                   │
│  Workflow Layer   Provider Layer    Workspace Layer             │
│  Data Layer       Search Engine     TOON Encoder                │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 Primary Constraint: IPC Boundary

The **IPC Boundary** between Rust (source of truth) and React (projection) is the primary architectural constraint. The system optimizes for:

1. **Tiered Streaming** — Ring buffer → 50ms debounced emit → `requestAnimationFrame` render
2. **Non-Blocking Supervision** — MCP process health loop runs independently; approval gate via oneshot channels
3. **Graph-Based Orchestration** — Petgraph DiGraph for workflow DAG execution

### 1.4 Architecturally Significant Requirements (ASRs)

| ASR ID       | Requirement                  | Architecture Impact                              |
| ------------ | ---------------------------- | ------------------------------------------------ |
| ASR-PERF-001 | Message render < 100ms (p99) | Ring buffer + 50ms debounce + rAF                |
| ASR-PERF-002 | UI maintains 60fps           | All blocking ops off main thread; virtualization |
| ASR-REL-001  | No data loss on crash        | SQLite WAL mode                                  |
| ASR-REL-002  | MCP supervision              | Health loop + exponential backoff restart        |
| ASR-SEC-001  | API keys in keychain only    | Credential service; no plaintext                 |
| ASR-SEC-002  | Tool approval gates          | Oneshot channels + DashMap                       |
| ASR-STR-001  | All business logic in Rust   | `skilldeck-core` with zero Tauri deps            |
| ASR-STR-002  | IPC-only frontend            | Command pattern; typed API layer                 |
| ASR-STR-003  | Multi-agent workflows        | Petgraph DAG + subagent forking                  |

//! Database connection management.
//!
//! This module handles SQLite database initialization, connection pooling,
//! and migration execution.

use sea_orm::{Database, DatabaseConnection, ConnectOptions};
use sea_orm_migration::MigratorTrait;
use std::time::Duration;
use tracing::info;

use crate::CoreError;
use super::migrations::Migrator;

/// Opens a SQLite database connection with WAL mode enabled.
///
/// # Arguments
///
/// * `url` - Database URL. Use `:memory:` for in-memory database, or a file path.
/// * `run_migrations` - Whether to run migrations on startup.
///
/// # Returns
///
/// A `DatabaseConnection` ready for use.
///
/// # Errors
///
/// Returns `CoreError::DatabaseConnection` if the connection cannot be established.
///
/// # Example
///
/// ```rust,no_run
/// let db = skilldeck_core::db::open_db(":memory:", true).await?;
/// ```
pub async fn open_db(url: &str, run_migrations: bool) -> Result<DatabaseConnection, CoreError> {
    let db_url = if url == ":memory:" {
        "sqlite::memory:".to_owned()
    } else {
        format!("sqlite://{}?mode=rwc", url)
    };

    let mut connect_options = ConnectOptions::new(db_url);
    connect_options
        .max_connections(5)
        .min_connections(1)
        .connect_timeout(Duration::from_secs(30))
        .acquire_timeout(Duration::from_secs(30))
        .idle_timeout(Some(Duration::from_secs(300)))
        .max_lifetime(Some(Duration::from_secs(1800)))
        .sqlx_logging(true)
        .sqlx_logging_level(tracing::log::LevelFilter::Debug);

    info!("Connecting to database: {}", if url == ":memory:" { "memory" } else { url });

    let db = Database::connect(connect_options)
        .await
        .map_err(|e| CoreError::DatabaseConnection {
            message: format!("Failed to connect: {}", e),
        })?;

    // Enable WAL mode for better concurrency
    db.execute(sea_orm::Statement::from_string(
        sea_orm::DatabaseBackend::Sqlite,
        "PRAGMA journal_mode=WAL".to_owned(),
    ))
    .await
    .map_err(|e| CoreError::DatabaseQuery {
        message: format!("Failed to enable WAL mode: {}", e),
    })?;

    // Enable foreign keys
    db.execute(sea_orm::Statement::from_string(
        sea_orm::DatabaseBackend::Sqlite,
        "PRAGMA foreign_keys=ON".to_owned(),
    ))
    .await
    .map_err(|e| CoreError::DatabaseQuery {
        message: format!("Failed to enable foreign keys: {}", e),
    })?;

    // Set busy timeout for better concurrency
    db.execute(sea_orm::Statement::from_string(
        sea_orm::DatabaseBackend::Sqlite,
        "PRAGMA busy_timeout=5000".to_owned(),
    ))
    .await

/// Trait for model providers.
///
/// Implementations provide access to AI model completions via streaming API.
#[async_trait]
pub trait ModelProvider: Send + Sync {
    /// Unique identifier for this provider.
    fn id(&self) -> &str;

    /// Human-readable display name.
    fn display_name(&self) -> &str;

    /// Whether this provider supports TOON format optimization.
    fn supports_toon(&self) -> bool {
        true
    }

    /// List available models for this provider.
    async fn list_models(&self) -> Result<Vec<ModelInfo>, CoreError>;

    /// Generate a completion stream.
    async fn complete(
        &self,
        request: CompletionRequest,
    ) -> Result<CompletionStream, CoreError>;

    /// Generate a single completion (non-streaming).
    async fn complete_sync(
        &self,
        request: CompletionRequest,
    ) -> Result<CompletionResult, CoreError> {
        use futures::StreamExt;

        let stream = self.complete(request).await?;
        let mut collected = stream.collect::<Vec<_>>().await;

        let mut content = String::new();
        let mut tool_calls = Vec::new();
        let mut usage = TokenUsage::default();

        for chunk in collected.into_iter().flatten() {
            match chunk {
                CompletionChunk::Token { content: token } => {
                    content.push_str(&token);
                }
                CompletionChunk::ToolCall { tool_call } => {
                    tool_calls.push(tool_call);
                }
                CompletionChunk::Done {
                    input_tokens,
                    output_tokens,
                    cache_read_tokens,
                    cache_write_tokens,
                } => {
                    usage = TokenUsage {
                        input_tokens,
                        output_tokens,
                        cache_read_tokens,
                        cache_write_tokens,
                    };
                }
            }
        }

        Ok(CompletionResult {
            content,
            tool_calls,
            usage,
            model: String::new(),
            finish_reason: if tool_calls.is_empty() {
                FinishReason::Stop
            } else {
                FinishReason::ToolCalls
            },
        })
    }
}

//! This module provides a trait that abstracts database access,

use async_trait::async_trait;

    /// Execute a raw SQL statement.

                .client
                .post(format!("{}/messages", self.base_url))
                .header("x-api-key", &self.api_key)
                .header("anthropic-version", "2023-06-01")
                .header("content-type", "application/json")
                .json(&claude_request)
                .send()
                .await
                .map_err(|e| {
                    warn!("Claude API request failed: {}", e);
                    backoff::Error::transient(CoreError::ModelConnection {
                        provider: "claude".to_string(),
                        message: e.to_string(),
                    })
                })?;

            let status = response.status();
            if !status.is_success() {
                let error_body = response.text().await.unwrap_or_default();
                warn!("Claude API error: {} - {}", status, error_body);

                return if status.as_u16() == 429 {
                    backoff::Error::transient(CoreError::ModelRateLimited {
                        provider: "claude".to_string(),
                        retry_after_ms: 1000,
                    })
                } else if status.is_server_error() {
                    backoff::Error::transient(CoreError::ModelInternal {
                        provider: "claude".to_string(),
                        message: format!("{}: {}", status, error_body),
                    })
                } else {
                    backoff::Error::permanent(CoreError::ModelRequestRejected {
                        provider: "claude".to_string(),
                        message: format!("{}: {}", status, error_body),
                    })
                };
            }

            Ok(response)
        };

        let response = retry(
            ExponentialBackoff {
                initial_interval: Duration::from_millis(100),
                max_interval: Duration::from_secs(10),
                multiplier: 2.0,
                max_elapsed_time: Some(Duration::from_secs(60)),
                ..Default::default()
            },
            operation,
        )
        .await?;

        // Create the streaming response
        let stream = response
            .bytes_stream()
            .filter_map(move |result| async move {
                match result {
                    Ok(bytes) => {
                        let text = String::from_utf8_lossy(&bytes);
                        let mut chunks = Vec::new();

                        for line in text.lines() {
                            if let Some(data) = line.strip_prefix("data: ") {
                                if data == "[DONE]" {
                                    continue;
                                }

                                if let Ok(event) = serde_json::from_str::<ClaudeEvent>(data) {
                                    if let Some(delta) = &event.delta {
                                        if let Some(text) = &delta.text {
                                            if !text.is_empty() {
                                                chunks.push(Ok(CompletionChunk::Token {
                                                    content: text.clone(),
                                                }));
                                            }
                                        }
                                    }

                                    if event.event_type == "message_stop" {
                                        if let Some(usage) = &event.message.as_ref().and_then(|m| m.usage.as_ref()) {
                                            chunks.push(Ok(CompletionChunk::Done {
                                                input_tokens: usage.input_tokens,
                                                output_tokens: usage.output_tokens,
                                                cache_read_tokens: usage.cache_read_input_tokens.unwrap_or(0),
                                                cache_write_tokens: usage.cache_creation_input_tokens.unwrap_or(0),
                                            }));
                                        }
                                    }
                                }
                            }
                        }

                        if chunks.len() == 1 {
                            Some(chunks.into_iter().next().unwrap())
                        } else if !chunks.is_empty() {
                            // Return first chunk, rest will be handled by subsequent iterations
                            Some(chunks.into_iter().next().unwrap())
                        } else {
                            None
                        }
                    }
                    Err(e) => Some(Err(CoreError::ModelConnection {
                        provider: "claude".to_string(),
                        message: e.to_string(),
                    })),
                }
            });

        Ok(Box::pin(stream))
    }

            stream_options: Some(json!({"include_usage": true})),
            tools,
        };

        let operation = || async {
            let response = self
                .client
                .post(format!("{}/chat/completions", self.base_url))
                .header("Authorization", format!("Bearer {}", self.api_key))
                .header("Content-Type", "application/json")
                .json(&openai_request)
                .send()
                .await
                .map_err(|e| {
                    warn!("OpenAI API request failed: {}", e);
                    backoff::Error::transient(CoreError::ModelConnection {
                        provider: "openai".to_string(),
                        message: e.to_string(),
                    })
                })?;

            let status = response.status();
            if !status.is_success() {
                let error_body = response.text().await.unwrap_or_default();
                warn!("OpenAI API error: {} - {}", status, error_body);

                return if status.as_u16() == 429 {
                    let retry_after = response
                        .headers()
                        .get("retry-after")
                        .and_then(|h| h.to_str().ok())
                        .and_then(|s| s.parse::<u64>().ok())
                        .unwrap_or(1);
                    backoff::Error::transient(CoreError::ModelRateLimited {
                        provider: "openai".to_string(),
                        retry_after_ms: retry_after * 1000,
                    })
                } else if status.is_server_error() {
                    backoff::Error::transient(CoreError::ModelInternal {
                        provider: "openai".to_string(),
                        message: format!("{}: {}", status, error_body),
                    })
                } else {
                    backoff::Error::permanent(CoreError::ModelRequestRejected {
                        provider: "openai".to_string(),
                        message: format!("{}: {}", status, error_body),
                    })
                };
            }

            Ok(response)
        };

        let response = retry(
            ExponentialBackoff {
                initial_interval: Duration::from_millis(100),
                max_interval: Duration::from_secs(10),
                multiplier: 2.0,
                max_elapsed_time: Some(Duration::from_secs(60)),
                ..Default::default()
            },
            operation,
        )
        .await?;

        let stream = response.bytes_stream().filter_map(move |result| async move {
            match result {
                Ok(bytes) => {
                    let text = String::from_utf8_lossy(&bytes);

                    for line in text.lines() {
                        if let Some(data) = line.strip_prefix("data: ") {
                            if data == "[DONE]" {
                                continue;
                            }

                            if let Ok(chunk) = serde_json::from_str::<OpenAiChunk>(data) {
                                if let Some(choice) = chunk.choices.first() {
                                    if let Some(content) = &choice.delta.content {
                                        if !content.is_empty() {
                                            return Some(Ok(CompletionChunk::Token {
                                                content: content.clone(),
                                            }));
                                        }
                                    }

                                    if let Some(tool_calls) = &choice.delta.tool_calls {
                                        if let Some(tc) = tool_calls.first() {
                                            return Some(Ok(CompletionChunk::ToolCall {
                                                tool_call: ToolCall {
                                                    id: tc.id.clone(),
                                                    r#type: tc.r#type.clone(),
                                                    function: tc.function.clone(),
                                                },
                                            }));
                                        }
                                    }
                                }

                                if let Some(usage) = &chunk.usage {
                                    return Some(Ok(CompletionChunk::Done {
                                        input_tokens: usage.prompt_tokens,
                                        output_tokens: usage.completion_tokens,
                                        cache_read_tokens: usage
                                            .prompt_tokens_details
                                            .as_ref()
                                            .map(|d| d.cached_tokens)
                                            .unwrap_or(0),
                                        cache_write_tokens: 0,
                                    }));
                                }
                            }
                        }
                    }
                    None
                }
                Err(e) => Some(Err(CoreError::ModelConnection {
                    provider: "openai".to_string(),
                    message: e.to_string(),
                })),
            }
        });

        Ok(Box::pin(stream))
    }

//! Filesystem-based skill loader.
//!
//! Parses SKILL.md files with YAML frontmatter.

use async_trait::async_trait;
use std::path::{Path, PathBuf};
use tokio::fs;
use sha2::{Sha256, Digest};

use crate::{
    CoreError,
    traits::{Skill, SkillLoader, SkillSource, SkillManifest},
};

/// Filesystem skill loader.
pub struct FilesystemSkillLoader;

impl FilesystemSkillLoader {
    /// Parse a SKILL.md file content.
    pub fn parse(content: &str, path: PathBuf, source: String) -> Result<Skill, CoreError> {
        // Remove BOM if present
        let content = content.trim_start_matches('\u{FEFF}');

        // Check for frontmatter delimiter
        if !content.starts_with("---") {
            return Err(CoreError::SkillParse {
                name: path.display().to_string(),
                message: "SKILL.md missing YAML frontmatter delimiter".into(),
            });
        }

        // Find closing delimiter
        let rest = &content[3..];
        let end = rest.find("\n---").ok_or_else(|| {
            CoreError::SkillParse {
                name: path.display().to_string(),
                message: "SKILL.md missing closing frontmatter delimiter".into(),
            }
        })?;

        let frontmatter = &rest[..end];
        let body = rest[end + 4..].trim_start_matches('\n');

        // Parse YAML frontmatter
        let manifest: SkillManifest = serde_yaml::from_str(frontmatter)
            .map_err(|e| CoreError::SkillParse {
                name: path.display().to_string(),
                message: format!("Invalid YAML frontmatter: {}", e),
            })?;

        // Extract name and description
        let name = manifest.extra.get("name")
            .and_then(|v| v.as_str())
            .ok_or_else(|| CoreError::SkillParse {
                name: path.display().to_string(),
                message: "Skill missing 'name' in frontmatter".into(),
            })?
            .to_string();

        let description = manifest.extra.get("description")
            .and_then(|v| v.as_str())
            .unwrap_or("")
            .to_string();

        // Compute content hash
        let content_hash = {
            let mut hasher = Sha256::new();
            hasher.update(body.as_bytes());
            format!("{:x}", hasher.finalize())
        };

        Ok(Skill {
            name,
            description,
            content_md: body.to_string(),
            manifest,
            disk_path: Some(path),
            source,
            content_hash: Some(content_hash),
        })
    }

    /// Compute hash for a skill directory.
    pub fn compute_dir_hash(skill_dir: &Path) -> String {
        let mut hasher = Sha256::new();
        // For now, just hash the path; in production, hash all files
        hasher.update(skill_dir.to_string_lossy().as_bytes());
        format!("{:x}", hasher.finalize())
    }
}

#[async_trait]
impl SkillLoader for FilesystemSkillLoader {
    async fn load(&self, source: &SkillSource) -> Result<Skill, CoreError> {
        match source {
            SkillSource::Filesystem(path) => {
                let skill_md = path.join("SKILL.md");

                if !skill_md.exists() {
                    return Err(CoreError::SkillNotFound { path: skill_md.clone() });
                }

                let content = fs::read_to_string(&skill_md).await?;
                Self::parse(&content, path.clone(), "filesystem".to_string())
            }
            SkillSource::Database(_) => Err(CoreError::NotImplemented {
                feature: "Database skill loading".into(),
            }),
            SkillSource::Remote(_) => Err(CoreError::NotImplemented {
                feature: "Remote skill loading".into(),
            }),
        }
    }

    async fn exists(&self, source: &SkillSource) -> bool {
        match source {
            SkillSource::Filesystem(path) => path.join("SKILL.md").exists(),
            _ => false,
        }
    }

    async fn modified_at(&self, source: &SkillSource) -> Option<std::time::SystemTime> {
        match source {
            SkillSource::Filesystem(path) => {
                let skill_md = path.join("SKILL.md");
                fs::metadata(&skill_md).await.ok()?.modified().ok()
            }
            _ => None,
        }
    }

//! Skill resolution with priority ordering.
//!
//! Resolves skill name conflicts using source priority:
//! workspace > personal > superpowers > marketplace

use std::collections::HashMap;
use crate::traits::Skill;

/// Source priority order (lower number = higher priority).
const SOURCE_PRIORITY: &[&str] = &["workspace", "personal", "superpowers", "marketplace"];

/// Result of skill resolution.
#[derive(Debug)]
pub struct ResolvedSkills {
    /// Skills that were selected.
    pub skills: Vec<Skill>,
    /// Names that were shadowed by higher-priority sources.
    pub shadowed: Vec<ShadowedSkill>,
}

/// Information about a shadowed skill.
#[derive(Debug)]
pub struct ShadowedSkill {
    pub name: String,
    pub source: String,
    pub shadowed_by: String,
}

/// Resolve skills from multiple sources by priority.
pub fn resolve(sources: Vec<(String, Vec<Skill>)>) -> ResolvedSkills {
    let mut by_name: HashMap<String, (Skill, String, usize)> = HashMap::new();
    let mut shadowed = Vec::new();

    // Sort sources by priority
    let mut sorted_sources: Vec<_> = sources.into_iter().collect();
    sorted_sources.sort_by_key(|(source_type, _)| {
        SOURCE_PRIORITY
            .iter()
            .position(|&s| s == source_type.as_str())
            .unwrap_or(999)
    });

    for (source_type, skills) in sorted_sources {
        let priority = SOURCE_PRIORITY
            .iter()
            .position(|&s| s == source_type.as_str())
            .unwrap_or(999);

        for skill in skills {
            if let Some((existing, existing_source, existing_priority)) = by_name.get(&skill.name) {
                if priority < *existing_priority {
                    // New skill has higher priority
                    shadowed.push(ShadowedSkill {
                        name: skill.name.clone(),
                        source: existing_source.clone(),
                        shadowed_by: source_type.clone(),
                    });
                    by_name.insert(skill.name.clone(), (skill, source_type, priority));
                } else {
                    // Existing skill has higher or equal priority
                    shadowed.push(ShadowedSkill {
                        name: skill.name.clone(),
                        source: skill.source.clone(),
                        shadowed_by: existing_source.clone(),
                    });
                }
            } else {
                by_name.insert(skill.name.clone(), (skill, source_type, priority));
            }
        }
    }

    let skills: Vec<_> = by_name.into_values().map(|(s, _, _)| s).collect();

    ResolvedSkills { skills, shadowed }
}

//! Filesystem watcher for skill directories.
//!
//! Detects skill file changes with 200ms debouncing.

use notify::{Event, EventKind, RecursiveMode, Watcher as NotifyWatcher};
use notify_debouncer_mini::{new_debouncer, DebouncedEvent};
use std::path::PathBuf;
use std::sync::Arc;
use std::time::Duration;
use tokio::sync::mpsc::Sender;
use tracing::{info, warn};

use crate::CoreError;

/// Events emitted by the skill watcher.
#[derive(Debug, Clone)]
pub enum SkillWatchEvent {
    /// A skill was created.
    Created(PathBuf),
    /// A skill was modified.
    Modified(PathBuf),
    /// A skill was deleted.
    Deleted(PathBuf),
}

/// Start watching a skill directory.
pub fn start_watcher(
    dir: PathBuf,
    tx: Sender<SkillWatchEvent>,
) -> Result<impl NotifyWatcher, CoreError> {
    let (debounce_tx, debounce_rx) = std::sync::mpsc::channel();

    let mut debouncer = new_debouncer(Duration::from_millis(200), None, debounce_tx)
        .map_err(|e| CoreError::Internal {
            message: format!("Failed to create debouncer: {}", e),
        })?;

    debouncer
        .watcher()
        .watch(&dir, RecursiveMode::Recursive)
        .map_err(|e| CoreError::FileOperation {
            path: dir.clone(),
            message: format!("Failed to start watcher: {}", e),
        })?;

    info!("Started skill watcher for {:?}", dir);

    // Spawn a thread to process debounced events
    std::thread::spawn(move || {
        while let Ok(events) = debounce_rx.recv() {
            if let Ok(events) = events {
                for DebouncedEvent { path, kind } in events {
                    // Only watch SKILL.md files
                    if path.file_name().map(|n| n == "SKILL.md").unwrap_or(false) {
                        // Get the skill directory (parent of SKILL.md)
                        if let Some(skill_dir) = path.parent() {
                            let event = match kind {
                                notify::EventKind::Create(_) => SkillWatchEvent::Created(skill_dir.to_owned()),
                                notify::EventKind::Modify(_) => SkillWatchEvent::Modified(skill_dir.to_owned()),
                                notify::EventKind::Remove(_) => SkillWatchEvent::Deleted(skill_dir.to_owned()),
                                _ => continue,
                            };

                            if tx.blocking_send(event).is_err() {
                                warn!("Failed to send skill watch event");
                            }
                        }
                    }
                }
            }
        }
    });

    Ok(debouncer)
}

//! Stdio transport for MCP servers.
//!
//! Communicates with MCP servers via subprocess stdin/stdout.

use async_trait::async_trait;
use serde_json::Value;
use std::process::Stdio;
use std::sync::atomic::{AtomicU64, Ordering};
use std::sync::Arc;
use tokio::io::{AsyncBufReadExt, AsyncWriteExt, BufReader};
use tokio::process::{Child, ChildStdin, ChildStdout, Command};
use tokio::sync::{mpsc, Mutex};
use tracing::{debug, error, info, warn};

use crate::{
    CoreError,
    traits::{McpTransport, McpServerConfig, McpSession, McpTool, McpToolResult, McpContent, McpCapabilities},
    mcp::types::*,
};

/// Stdio MCP transport.
pub struct StdioTransport;

impl StdioTransport {
    /// Parse server config into command and args.
    fn parse_config(config: &Value) -> Result<(String, Vec<String>), CoreError> {
        let command = config["command"]
            .as_str()
            .ok_or_else(|| CoreError::InvalidConfiguration {
                message: "Missing 'command' in stdio config".to_string(),
            })?
            .to_string();

        let args = config["args"]
            .as_array()
            .map(|arr| {
                arr.iter()
                    .filter_map(|v| v.as_str().map(String::from))
                    .collect()
            })
            .unwrap_or_default();

        Ok((command, args))
    }
}

#[async_trait]
impl McpTransport for StdioTransport {
    async fn connect(
        &self,
        config: &McpServerConfig,
        server_name: &str,
    ) -> Result<McpSession, CoreError> {
        let (command, args) = Self::parse_config(&config.config)?;

        info!("Starting MCP server '{}': {} {:?}", server_name, command, args);

        let mut child = Command::new(&command)
            .args(&args)
            .stdin(Stdio::piped())
            .stdout(Stdio::piped())
            .stderr(Stdio::piped())
            .spawn()
            .map_err(|e| CoreError::McpConnectionFailed {
                server_name: server_name.to_string(),
                message: format!("Failed to spawn process: {}", e),
            })?;

        let stdin = child.stdin.take().ok_or_else(|| CoreError::McpConnectionFailed {
            server_name: server_name.to_string(),
            message: "Failed to open stdin".to_string(),
        })?;

        let stdout = child.stdout.take().ok_or_else(|| CoreError::McpConnectionFailed {
            server_name: server_name.to_string(),
            message: "Failed to open stdout".to_string(),
        })?;

        let inner = StdioSessionInner::new(stdin, stdout, child);

        // Initialize the session
        let capabilities = inner.initialize().await?;

        // List tools
        let tools = inner.list_tools().await?;

        Ok(McpSession {
            server_name: server_name.to_string(),
            tools,
            capabilities,
            inner: Box::new(inner),
        })
    }

    fn supports(&self, config: &McpServerConfig) -> bool {
        config.transport == "stdio"
    }
}

/// Inner stdio session implementation.
struct StdioSessionInner {
    stdin: Arc<Mutex<ChildStdin>>,
    stdout: Arc<Mutex<BufReader<ChildStdout>>>,
    child: Arc<Mutex<Option<Child>>>,
    request_id: AtomicU64,
}

impl StdioSessionInner {
    fn new(stdin: ChildStdin, stdout: ChildStdout, child: Child) -> Self {
        Self {
            stdin: Arc::new(Mutex::new(stdin)),
            stdout: Arc::new(Mutex::new(BufReader::new(stdout))),
            child: Arc::new(Mutex::new(Some(child))),
            request_id: AtomicU64::new(1),
        })
    }

    async fn send_request(&self, method: &str, params: Option<Value>) -> Result<JsonRpcResponse, CoreError> {
        let id = self.request_id.fetch_add(1, Ordering::SeqCst);
        let request = JsonRpcRequest::new(id, method);
        let request = if let Some(p) = params {
            request.with_params(p)
        } else {
            request
        };

        let request_json = serde_json::to_string(&request)
            .map_err(|e| CoreError::Internal {
                message: format!("Failed to serialize request: {}", e),
            })?;

        debug!("Sending JSON-RPC request: {}", request_json);

        // Send request
        {
            let mut stdin = self.stdin.lock().await;
            stdin
                .write_all(request_json.as_bytes())
                .await
                .map_err(|e| CoreError::McpConnectionFailed {
                    server_name: String::new(),
                    message: format!("Failed to write to stdin: {}", e),
                })?;
            stdin
                .write_all(b"\n")
                .await
                .map_err(|e| CoreError::McpConnectionFailed {
                    server_name: String::new(),
                    message: format!("Failed to write newline: {}", e),
                })?;
            stdin.flush().await.map_err(|e| CoreError::McpConnectionFailed {
                server_name: String::new(),
                message: format!("Failed to flush stdin: {}", e),
            })?;
        }

        // Read response
        let response_line = {
            let mut stdout = self.stdout.lock().await;
            let mut line = String::new();
            stdout
                .read_line(&mut line)
                .await
                .map_err(|e| CoreError::McpConnectionFailed {
                    server_name: String::new(),
                    message: format!("Failed to read from stdout: {}", e),
                })?;
            line
        };

        debug!("Received JSON-RPC response: {}", response_line.trim());

        let response: JsonRpcResponse = serde_json::from_str(response_line.trim())
            .map_err(|e| CoreError::McpJsonRpc {
                code: -32700,
                message: format!("Failed to parse response: {}", e),
            })?;

        if let Some(error) = &response.error {
            return Err(CoreError::McpJsonRpc {
                code: error.code,
                message: error.message.clone(),
            });
        }

        Ok(response)
    }

    async fn initialize(&self) -> Result<McpCapabilities, CoreError> {
        let params = serde_json::to_value(InitializeParams::default())
            .map_err(|e| CoreError::Internal {
                message: format!("Failed to serialize init params: {}", e),
            })?;

        let response = self.send_request("initialize", Some(params)).await?;

        let result: InitializeResult = response
            .result
            .ok_or_else(|| CoreError::McpJsonRpc {
                code: -32603,
                message: "No result in initialize response".to_string(),
            })?
            .deserialize()
            .map_err(|e| CoreError::McpJsonRpc {
                code: -32603,
                message: format!("Failed to parse initialize result: {}", e),
            })?;

        info!(
            "MCP server initialized: {} v{} (protocol {})",
            result.server_info.name,
            result.server_info.version,
            result.protocol_version
        );

        // Send initialized notification
        self.send_request("notifications/initialized", None).await.ok();

        Ok(McpCapabilities {
            tools: result.capabilities.tools.is_some(),
            resources: result.capabilities.resources.is_some(),
            prompts: result.capabilities.prompts.is_some(),
        })
    }

    async fn list_tools(&self) -> Result<Vec<McpTool>, CoreError> {
        let params = serde_json::to_value(ListToolsParams::default())
            .map_err(|e| CoreError::Internal {
                message: format!("Failed to serialize list_tools params: {}", e),
            })?;

        let response = self.send_request("tools/list", Some(params)).await?;

        let result: ListToolsResult = response
            .result
            .ok_or_else(|| CoreError::McpJsonRpc {
                code: -32603,
                message: "No result in list_tools response".to_string(),
            })?
            .deserialize()
            .map_err(|e| CoreError::McpJsonRpc {
                code: -32603,
                message: format!("Failed to parse list_tools result: {}", e),
            })?;

        Ok(result
            .tools
            .into_iter()
            .map(|t| McpTool {
                name: t.name,
                description: t.description,
                input_schema: t.input_schema,
            })
            .collect())
    }
}

/// Inner SSE session.
struct SseSessionInner {
    client: Client,
    base_url: String,
    server_name: String,
    request_id: AtomicU64,
}

impl SseSessionInner {
    fn new(client: Client, base_url: String, server_name: String) -> Self {
        Self {
            client,
            base_url,
            server_name,
            request_id: AtomicU64::new(1),
        }
    }

    async fn send_request(&self, method: &str, params: Option<Value>) -> Result<JsonRpcResponse, CoreError> {
        let id = self.request_id.fetch_add(1, Ordering::SeqCst);
        let request = JsonRpcRequest::new(id, method);
        let request = if let Some(p) = params {
            request.with_params(p)
        } else {
            request
        };

        let url = format!("{}/mcp", self.base_url);
        debug!("Sending SSE request to {}: {:?}", url, request);

        let response = self
            .client
            .post(&url)
            .json(&request)
            .send()
            .await
            .map_err(|e| CoreError::McpConnectionFailed {
                server_name: self.server_name.clone(),
                message: format!("HTTP request failed: {}", e),
            })?;

        if !response.status().is_success() {
            let status = response.status();
            let body = response.text().await.unwrap_or_default();
            return Err(CoreError::McpConnectionFailed {
                server_name: self.server_name.clone(),
                message: format!("HTTP {}: {}", status, body),
            });
        }

        // Parse SSE response
        let mut stream = response.bytes_stream();
        let mut body = String::new();

        while let Some(chunk) = stream.next().await {
            match chunk {
                Ok(bytes) => {
                    body.push_str(&String::from_utf8_lossy(&bytes));
                }
                Err(e) => {
                    return Err(CoreError::McpConnectionFailed {
                        server_name: self.server_name.clone(),
                        message: format!("Stream error: {}", e),
                    });
                }
            }

//! MCP server registry for managing connections.

use dashmap::DashMap;
use std::sync::Arc;
use tokio::sync::RwLock;
use uuid::Uuid;

use crate::{
    CoreError,
    traits::{McpSession, McpTool, McpCapabilities},

/// Server connection state.
#[derive(Debug, Clone, PartialEq, Eq)]
pub enum ServerStatus {
    Disconnected,
    Connecting,
    Connected,
    Error,
    Failed,
}

/// Live server session.
pub struct LiveServer {
    pub id: Uuid,
    pub name: String,
    pub status: ServerStatus,
    pub session: Option<Arc<McpSession>>,
    pub error_count: u32,
    pub last_error: Option<String>,
    pub tools: Vec<McpTool>,
    pub capabilities: McpCapabilities,
}

/// MCP server registry.
pub struct McpRegistry {
    servers: DashMap<Uuid, LiveServer>,
    transports: Vec<Box<dyn crate::traits::McpTransport>>,
}

impl McpRegistry {
    pub fn new() -> Self {
        Self {
            servers: DashMap::new(),
            transports: vec![],
        }
    }

    /// Register a transport.
    pub fn register_transport(&mut self, transport: impl crate::traits::McpTransport + 'static) {
        self.transports.push(Box::new(transport));
    }

    /// Add a server to the registry (not yet connected).
    pub fn add_server(&self, name: String, _config: McpServerConfig) -> Uuid {
        let id = Uuid::new_v4();
        self.servers.insert(
            id,
            LiveServer {
                id,
                name,
                status: ServerStatus::Disconnected,
                session: None,
                error_count: 0,
                last_error: None,
                tools: Vec::new(),
                capabilities: McpCapabilities::default(),
            },
        );
        id
    }

    /// Connect to a server.
    pub async fn connect(&self, id: Uuid, config: McpServerConfig) -> Result<(), CoreError> {
        let server = self
            .servers
            .get_mut(&id)
            .ok_or_else(|| CoreError::McpServerNotFound { server_id: id })?;

        server.status = ServerStatus::Connecting;

        // Find appropriate transport
        let transport = self
            .transports
            .iter()
            .find(|t| t.supports(&config))
            .ok_or_else(|| CoreError::InvalidConfiguration {
                message: format!("No transport for type: {}", config.transport),
            })?;

        match transport.connect(&config, &server.name).await {
            Ok(session) => {
                server.tools = session.tools.clone();
                server.capabilities = McpCapabilities {
                    tools: !session.tools.is_empty(),
                    resources: false,
                    prompts: false,
                };
                server.session = Some(Arc::new(session));
                server.status = ServerStatus::Connected;
                server.error_count = 0;
                server.last_error = None;
                Ok(())
            }
            Err(e) => {
                server.status = ServerStatus::Error;
                server.error_count += 1;
                server.last_error = Some(e.to_string());
                Err(e)
            }
        }
    }

    /// Disconnect a server.
    pub fn disconnect(&self, id: Uuid) {
        if let Some(mut server) = self.servers.get_mut(&id) {
            server.session = None;
            server.status = ServerStatus::Disconnected;
        }
    }

    /// Get server by ID.
    pub fn get(&self, id: Uuid) -> Option<LiveServer> {
        self.servers.get(&id).map(|s| s.clone())
    }

    /// List all servers.
    pub fn list(&self) -> Vec<LiveServer> {
        self.servers.iter().map(|s| s.clone()).collect()
    }

    /// Get all tools from connected servers.
    pub fn all_tools(&self) -> Vec<(String, McpTool)> {
        self.servers
            .iter()
            .filter(|s| s.status == ServerStatus::Connected)
            .flat_map(|s| {
                let name = s.name.clone();
                s.tools.iter().map(move |t| (name.clone(), t.clone()))
            })
            .collect()
    }

    /// Call a tool on a specific server.
    pub async fn call_tool(
        &self,
        server_name: &str,
        tool_name: &str,
        arguments: serde_json::Value,
    ) -> Result<crate::traits::McpToolResult, CoreError> {
        let server = self
            .servers
            .iter()
            .find(|s| s.name == server_name && s.status == ServerStatus::Connected)
            .ok_or_else(|| CoreError::McpToolNotFound {
                server_name: server_name.to_string(),
                tool_name: tool_name.to_string(),
            })?;

        let session = server
            .session
            .as_ref()
            .ok_or_else(|| CoreError::McpDisconnected {
                server_name: server_name.to_string(),
                message: "No active session".to_string(),
            })?;

        session.call_tool(tool_name, arguments).await
    }
}

impl Default for McpRegistry {
    fn default() -> Self {
        Self::new()
    }

//! MCP server supervisor for health monitoring and restart.

use std::sync::Arc;
use std::time::{Duration, Instant};
use tokio::sync::mpsc;
use tracing::{error, info, warn};

use crate::{CoreError, mcp::registry::{McpRegistry, ServerStatus}};

/// Supervisor configuration.
#[derive(Debug, Clone)]
pub struct SupervisorConfig {
    /// Health check interval.
    pub check_interval: Duration,
    /// Initial restart delay.
    pub initial_delay: Duration,
    /// Maximum restart delay.
    pub max_delay: Duration,
    /// Backoff multiplier.
    pub multiplier: f64,
    /// Maximum restart attempts before giving up.
    pub max_attempts: u32,
}

impl Default for SupervisorConfig {
    fn default() -> Self {
        Self {
            check_interval: Duration::from_secs(30),
            initial_delay: Duration::from_secs(1),
            max_delay: Duration::from_secs(60),
            multiplier: 2.0,
            max_attempts: 5,
        }
    }
}

/// Restart state for a server.
struct RestartState {
    attempts: u32,
    next_delay: Duration,
    last_attempt: Instant,
}

impl RestartState {
    fn new() -> Self {
        Self {
            attempts: 0,
            next_delay: Duration::from_secs(1),
            last_attempt: Instant::now(),
        }
    }

    fn record_failure(&mut self, config: &SupervisorConfig) -> Duration {
        self.attempts += 1;
        self.last_attempt = Instant::now();
        let delay = self.next_delay;
        self.next_delay = Duration::from_secs_f64(
            (self.next_delay.as_secs_f64() * config.multiplier).min(config.max_delay.as_secs_f64()),
        );
        delay
    }

    fn reset(&mut self) {
        self.attempts = 0;
        self.next_delay = Duration::from_secs(1);
    }
}

/// Start the supervisor loop.
pub fn start_supervisor(
    registry: Arc<McpRegistry>,
    config: SupervisorConfig,
) -> mpsc::Sender<SupervisorCommand> {
    let (tx, mut rx) = mpsc::channel(32);

    tokio::spawn(async move {
        let mut restart_states: std::collections::HashMap<uuid::Uuid, RestartState> =
            std::collections::HashMap::new();

        let mut interval = tokio::time::interval(config.check_interval);

        loop {
            tokio::select! {
                _ = interval.tick() => {
                    // Health check
                    for server in registry.list() {
                        if server.status == ServerStatus::Error {
                            let state = restart_states.entry(server.id).or_insert_with(RestartState::new);

                            if state.attempts >= config.max_attempts {
                                warn!(
                                    "Server {} exceeded max restart attempts ({})",
                                    server.name, config.max_attempts
                                );
                                continue;
                            }

                            if Instant::now().duration_since(state.last_attempt) >= state.next_delay {
                                info!("Attempting restart of server {}", server.name);

                                // Signal restart needed
                                let delay = state.record_failure(&config);
                                info!("Next restart in {:?}", delay);
                            }
                        } else if server.status == ServerStatus::Connected {
                            // Reset restart state on successful connection
                            if let Some(state) = restart_states.get_mut(&server.id) {
                                state.reset();
                            }
                        }
                    }
                }

                Some(cmd) = rx.recv() => {
                    match cmd {
                        SupervisorCommand::Stop => {
                            info!("Supervisor stopping");
                            break;
                        }
                        SupervisorCommand::Restart(server_id) => {
                            let server = registry.get(server_id);
                            if let Some(server) = server {
                                info!("Manual restart requested for {}", server.name);
                                // Reset restart state
                                if let Some(state) = restart_states.get_mut(&server_id) {
                                    state.reset();
                                }
                            }
                        }
                        SupervisorCommand::Reset(server_id) => {
                            if let Some(state) = restart_states.get_mut(&server_id) {
                                state.reset();
                            }
                        }
                    }
                }
            }
        }
    });

    tx
}

/// Commands to the supervisor.
#[derive(Debug)]
pub enum SupervisorCommand {
    /// Stop the supervisor.
    Stop,
    /// Request immediate restart.
    Restart(uuid::Uuid),
    /// Reset restart counter.
    Reset(uuid::Uuid),
}

//! Core agent loop implementation.
//!
//! The agent loop is the heart of the conversation engine:
//! 1. Receive user message
//! 2. Build context (history, skills, system prompt)
//! 3. Call model provider
//! 4. Stream response with 50ms debounce
//! 5. Handle tool calls (built-in or MCP)
//! 6. Repeat until done

use futures::StreamExt;
use std::sync::Arc;
use std::time::Duration;
use tokio::sync::{mpsc, oneshot};
use tokio::time::{interval, Instant};
use tracing::{debug, error, info, instrument};

use crate::{
    CoreError,
    traits::{
        ModelProvider, CompletionRequest, CompletionChunk, ChatMessage, MessageRole,
        ToolDefinition, ToolCall,
    },
    events::AgentEvent,
};

/// Agent loop configuration.
#[derive(Debug, Clone)]
pub struct AgentLoopConfig {
    /// Debounce interval for token emission.
    pub debounce_ms: u64,
    /// Maximum context messages to include.
    pub max_context_messages: usize,
    /// Maximum tool call iterations.
    pub max_tool_iterations: u32,
}

impl Default for AgentLoopConfig {
    fn default() -> Self {
        Self {
            debounce_ms: 50,
            max_context_messages: 100,
            max_tool_iterations: 10,
        }
    }
}

/// Running agent loop state.
pub struct AgentLoop {
    config: AgentLoopConfig,
    provider: Arc<dyn ModelProvider>,
    model_id: String,
    system_prompt: Option<String>,
    messages: Vec<ChatMessage>,
    skills: Vec<String>,
    tools: Vec<ToolDefinition>,
    tx: mpsc::Sender<Result<AgentEvent, CoreError>>,
}

impl AgentLoop {
    /// Create a new agent loop.
    pub fn new(
        provider: Arc<dyn ModelProvider>,
        model_id: String,
        config: AgentLoopConfig,
        tx: mpsc::Sender<Result<AgentEvent, CoreError>>,
    ) -> Self {
        Self {
            config,
            provider,
            model_id,
            system_prompt: None,
            messages: Vec::new(),
            skills: Vec::new(),
            tools: Vec::new(),
            tx,
        }
    }

    /// Set the system prompt.
    pub fn with_system_prompt(mut self, prompt: String) -> Self {
        self.system_prompt = Some(prompt);
        self
    }

    /// Add a skill.
    pub fn with_skill(mut self, skill_content: String) -> Self {
        self.skills.push(skill_content);
        self
    }

    /// Add a tool.
    pub fn with_tool(mut self, tool: ToolDefinition) -> Self {
        self.tools.push(tool);
        self
    }

    /// Set conversation history.
    pub fn with_history(mut self, messages: Vec<ChatMessage>) -> Self {
        self.messages = messages;
        self
    }

    /// Run the agent loop with a user message.
    #[instrument(skip(self))]
    pub async fn run(mut self, user_message: String) -> Result<(), CoreError> {
        info!("Starting agent loop");

        // Add user message
        self.messages.push(ChatMessage {
            role: MessageRole::User,
            content: user_message,
            name: None,
        });

        let mut iteration = 0;

        loop {
            iteration += 1;
            if iteration > self.config.max_tool_iterations {
                warn!("Max tool iterations reached");
                break;
            }

            // Build request
            let request = self.build_request()?;

            // Call provider
            let stream = self.provider.complete(request).await?;

            // Process stream
            let result = self.process_stream(stream).await?;

            // Add assistant message
            self.messages.push(ChatMessage {
                role: MessageRole::Assistant,
                content: result.content.clone(),
                name: None,
            });

            // Handle tool calls
            if result.tool_calls.is_empty() {
                // Done
                let _ = self.tx.send(Ok(AgentEvent::Done {
                    input_tokens: result.input_tokens,
                    output_tokens: result.output_tokens,
                    cache_read_tokens: result.cache_read_tokens,
                    cache_write_tokens: result.cache_write_tokens,
                })).await;
                break;
            }

            // Process tool calls
            for tool_call in result.tool_calls {
                let _ = self.tx.send(Ok(AgentEvent::ToolCall {
                    tool_call: tool_call.clone(),
                })).await;

                // Execute tool
                let tool_result = self.execute_tool(&tool_call).await?;

                // Add tool result to messages
                self.messages.push(ChatMessage {
                    role: MessageRole::Tool,
                    content: serde_json::to_string(&tool_result).unwrap_or_default(),
                    name: Some(tool_call.function.name.clone()),
                });
            }
        }

        info!("Agent loop completed");
        Ok(())
    }

    fn build_request(&self) -> Result<CompletionRequest, CoreError> {
        // Build system prompt with skills
        let mut system_parts = Vec::new();

        if let Some(ref prompt) = self.system_prompt {
            system_parts.push(prompt.clone());
        }

        for skill in &self.skills {
            system_parts.push(skill.clone());
        }

        let system = if system_parts.is_empty() {
            None
        } else {
            Some(system_parts.join("\n\n---\n\n"))
        };

        // Limit context
        let messages = if self.messages.len() > self.config.max_context_messages {
            let skip = self.messages.len() - self.config.max_context_messages;
            self.messages.iter().skip(skip).cloned().collect()
        } else {
            self.messages.clone()
        };

        Ok(CompletionRequest {
            messages,
            system,
            tools: self.tools.clone(),
            model_params: Default::default(),
            model_id: self.model_id.clone(),

    async fn process_stream(
        &self,
        mut stream: crate::traits::CompletionStream,
    ) -> Result<StreamResult, CoreError> {
        let mut content = String::new();
        let mut tool_calls = Vec::new();
        let mut input_tokens = 0u32;
        let mut output_tokens = 0u32;
        let mut cache_read_tokens = 0u32;
        let mut cache_write_tokens = 0u32;

        // Debounce state
        let mut buffer = String::new();
        let mut last_emit = Instant::now();
        let debounce = Duration::from_millis(self.config.debounce_ms);

        while let Some(chunk) = stream.next().await {
            match chunk {
                Ok(CompletionChunk::Token { content: token }) => {
                    content.push_str(&token);
                    buffer.push_str(&token);

                    // Debounce emission
                    if last_emit.elapsed() >= debounce || buffer.len() > 100 {
                        let _ = self.tx.send(Ok(AgentEvent::Token {
                            delta: buffer.clone(),
                        })).await;
                        buffer.clear();
                        last_emit = Instant::now();
                    }
                }
                Ok(CompletionChunk::ToolCall { tool_call }) => {
                    // Flush buffer
                    if !buffer.is_empty() {
                        let _ = self.tx.send(Ok(AgentEvent::Token {
                            delta: buffer.clone(),
                        })).await;
                        buffer.clear();
                    }

                    tool_calls.push(tool_call);
                }
                Ok(CompletionChunk::Done {
                    input_tokens: it,
                    output_tokens: ot,
                    cache_read_tokens: crt,
                    cache_write_tokens: cwt,
                }) => {
                    input_tokens = it;
                    output_tokens = ot;
                    cache_read_tokens = crt;
                    cache_write_tokens = cwt;
                }
                Err(e) => {
                    error!("Stream error: {}", e);
                    let _ = self.tx.send(Err(e)).await;
                }
            }
        }

        // Flush remaining buffer
        if !buffer.is_empty() {
            let _ = self.tx.send(Ok(AgentEvent::Token { delta: buffer })).await;
        }

        Ok(StreamResult {
            content,
            tool_calls,
            input_tokens,
            output_tokens,
            cache_read_tokens,
            cache_write_tokens,
        })
    }

//! Tool dispatcher for routing tool calls.
//!
//! Routes tool calls to:
//! - Built-in tools (loadSkill, spawnSubagent, mergeSubagentResults)
//! - MCP servers
//! - Approval gates for external tools

use std::sync::Arc;

use crate::{
    CoreError,
    traits::{ToolCall, McpToolResult, McpContent},
    mcp::registry::McpRegistry,

/// Tool dispatcher.
pub struct ToolDispatcher {
    mcp_registry: Arc<McpRegistry>,
    approval_gate: ApprovalGate,
}

impl ToolDispatcher {
    pub fn new(mcp_registry: Arc<McpRegistry>, approval_gate: ApprovalGate) -> Self {
        Self {
            mcp_registry,
            approval_gate,
        }
    }

    /// Dispatch a tool call.
    pub async fn dispatch(&self, tool_call: &ToolCall) -> Result<Value, CoreError> {
        let tool_name = &tool_call.function.name;
        let arguments: Value = serde_json::from_str(&tool_call.function.arguments)
            .unwrap_or(Value::Null);

        // Check if it's a built-in tool
        if let Some(result) = self.dispatch_builtin(tool_name, &arguments).await {
            return result;
        }

        // Check if approval is needed
        if self.needs_approval(tool_name) {
            let approval = self.approval_gate.request_approval(
                tool_call.id.clone(),
                tool_name.clone(),
                arguments.clone(),
            ).await?;

            match approval {
                ApprovalResult::Approved { edited_input } => {
                    self.dispatch_mcp(tool_name, &edited_input.unwrap_or(arguments)).await
                }
                ApprovalResult::Denied { reason } => {
                    Err(CoreError::McpToolExecution {
                        tool_name: tool_name.clone(),
                        message: reason.unwrap_or_else(|| "Denied by user".to_string()),
                    })
                }
                ApprovalResult::Cancelled => {
                    Err(CoreError::Cancelled {
                        operation: format!("tool:{}", tool_name),
                    })
                }
            }
        } else {
            self.dispatch_mcp(tool_name, &arguments).await
        }
    }

    async fn dispatch_builtin(
        &self,
        tool_name: &str,
        arguments: &Value,
    ) -> Option<Result<Value, CoreError>> {
        match tool_name {
            "loadSkill" => {
                // Built-in: load a skill
                let skill_name = arguments["name"].as_str()?;
                Some(Ok(serde_json::json!({
                    "loaded": skill_name
                })))
            }
            "spawnSubagent" => {
                // Built-in: spawn a subagent
                Some(Ok(serde_json::json!({
                    "spawned": true
                })))
            }
            "mergeSubagentResults" => {
                // Built-in: merge subagent results
                Some(Ok(serde_json::json!({
                    "merged": true
                })))
            }
            _ => None,
        }
    }

    async fn dispatch_mcp(
        &self,
        tool_name: &str,
        arguments: &Value,
    ) -> Result<Value, CoreError> {
        // Find which server has this tool
        let tools = self.mcp_registry.all_tools();
        let (server_name, _) = tools
            .iter()
            .find(|(_, tool)| tool.name == tool_name)
            .ok_or_else(|| CoreError::McpToolNotFound {
                server_name: String::new(),
                tool_name: tool_name.to_string(),
            })?;

        let result = self.mcp_registry
            .call_tool(server_name, tool_name, arguments.clone())
            .await?;

        // Convert result to JSON
        Ok(serde_json::to_value(&result).unwrap_or(Value::Null))
    }

    fn needs_approval(&self, tool_name: &str) -> bool {
        // Built-in tools don't need approval
        !matches!(
            tool_name,
            "loadSkill" | "spawnSubagent" | "mergeSubagentResults"
        )
    }
}

/// Approval gate for tool calls.
pub struct ApprovalGate {
    pending: dashmap::DashMap<String, oneshot::Sender<ApprovalResult>>,
}

impl ApprovalGate {
    pub fn new() -> Self {
        Self {
            pending: dashmap::DashMap::new(),
        }
    }

    /// Request approval for a tool call.
    pub async fn request_approval(
        &self,
        tool_call_id: String,
        tool_name: String,
        input: Value,
    ) -> Result<ApprovalResult, CoreError> {
        let (tx, rx) = oneshot::channel();

        self.pending.insert(tool_call_id.clone(), tx);

        // Wait for approval
        rx.await.map_err(|_| CoreError::Cancelled {
            operation: format!("tool approval:{}", tool_call_id),
        })
    }

    /// Resolve an approval request.
    pub fn resolve(
        &self,
        tool_call_id: &str,
        result: ApprovalResult,
    ) -> Result<(), CoreError> {
        if let Some((_, tx)) = self.pending.remove(tool_call_id) {
            let _ = tx.send(result);
            Ok(())
        } else {
            Err(CoreError::Internal {
                message: format!("No pending approval for {}", tool_call_id),
            })
        }
    }

    /// Cancel all pending approvals.
    pub fn cancel_all(&self) {
        for entry in self.pending.iter() {
            let _ = entry.send(ApprovalResult::Cancelled);
        }
        self.pending.clear();
    }
}

impl Default for ApprovalGate {
    fn default() -> Self {
        Self::new()
    }
}

//! Workflow DAG implementation using petgraph.

use petgraph::{Directed, Graph, algo::toposort, graph::NodeIndex};
use std::collections::HashMap;

use super::types::*;
use crate::CoreError;

/// Workflow graph wrapper.
pub struct WorkflowGraph {
    graph: Graph<String, ()>, // Node = step ID
    step_indices: HashMap<String, NodeIndex>,
}

impl WorkflowGraph {
    pub fn new() -> Self {
        Self {
            graph: Graph::new(),
            step_indices: HashMap::new(),
        }
    }

    /// Build graph from definition.
    pub fn from_definition(def: &WorkflowDefinition) -> Result<Self, CoreError> {
        let mut graph = Self::new();

        // Add all steps as nodes
        for step in &def.steps {
            graph.add_step(&step.id);
        }

        // Add dependencies as edges
        for dep in &def.dependencies {
            graph.add_dependency(&dep.from, &dep.to)?;
        }

        // Check for cycles
        graph.validate()?;

        Ok(graph)
    }

    fn add_step(&mut self, step_id: &str) {
        let idx = self.graph.add_node(step_id.to_string());
        self.step_indices.insert(step_id.to_string(), idx);
    }

    fn add_dependency(&mut self, from: &str, to: &str) -> Result<(), CoreError> {
        let from_idx = self.step_indices.get(from).copied().ok_or_else(|| {
            CoreError::WorkflowInvalidDefinition {
                message: format!("Unknown step: {}", from),
            }
        })?;

        let to_idx = self.step_indices.get(to).copied().ok_or_else(|| {
            CoreError::WorkflowInvalidDefinition {
                message: format!("Unknown step: {}", to),
            }
        })?;

        self.graph.add_edge(from_idx, to_idx, ());

        Ok(())
    }

    /// Validate graph (no cycles).
    pub fn validate(&self) -> Result<(), CoreError> {
        match toposort(&self.graph, None) {
            Ok(_) => Ok(()),
            Err(_) => Err(CoreError::WorkflowCycle {
                cycle_path: "Cycle detected in workflow".to_string(),
            }),
        }
    }

    /// Get execution order (topological sort).
    pub fn execution_order(&self) -> Result<Vec<String>, CoreError> {
        let sorted = toposort(&self.graph, None).map_err(|_| CoreError::WorkflowCycle {
            cycle_path: "Cycle detected".to_string(),
        })?;

        Ok(sorted
            .into_iter()
            .map(|idx| self.graph[idx].clone())
            .collect())
    }

    /// Get dependencies of a step.
    pub fn dependencies(&self, step_id: &str) -> Vec<String> {
        let Some(&idx) = self.step_indices.get(step_id) else {
            return vec![];
        };

        self.graph
            .neighbors_directed(idx, petgraph::Direction::Incoming)
            .map(|dep_idx| self.graph[dep_idx].clone())
            .collect()
    }

    /// Get dependents of a step.
    pub fn dependents(&self, step_id: &str) -> Vec<String> {
        let Some(&idx) = self.step_indices.get(step_id) else {
            return vec![];
        };

        self.graph
            .neighbors_directed(idx, petgraph::Direction::Outgoing)
            .map(|dep_idx| self.graph[dep_idx].clone())
            .collect()
    }
}

//! Workspace context loading.
//!
//! Loads context files (CLAUDE.md, README, etc.) for AI context.

use std::path::{Path, PathBuf};
use tokio::fs;
use tracing::{debug, info, warn};

use crate::CoreError;
use super::detector::ProjectType;

/// Loaded workspace context.
#[derive(Debug, Clone)]
pub struct WorkspaceContext {
    /// Workspace root path.
    pub root: PathBuf,
    /// Project type.
    pub project_type: ProjectType,
    /// Loaded context files.
    pub context_files: Vec<ContextFile>,
    /// Skill directory path.
    pub skill_directory: Option<PathBuf>,
    /// Whether this is a git repository.
    pub is_git_repo: bool,
    /// Gitignore patterns.
    pub gitignore_patterns: Vec<String>,
}

/// A loaded context file.
#[derive(Debug, Clone)]
pub struct ContextFile {
    pub name: String,
    pub path: PathBuf,
    pub content: String,
    pub priority: u32,
}

/// Context loader.
pub struct ContextLoader;

impl ContextLoader {
    /// Load workspace context from a directory.
    pub async fn load<P: AsRef<Path>>(root: P) -> Result<WorkspaceContext, CoreError> {
        let root = root.as_ref().to_path_buf();

        // Detect project type
        let project_type = super::detector::WorkspaceDetector::detect(&root).await;

        info!("Loading context for {:?} ({})", root, project_type);

        // Load context files
        let context_files = Self::load_context_files(&root, project_type).await?;

        // Check for skill directory
        let skill_directory = Self::find_skill_directory(&root).await;

        // Check for git
        let is_git_repo = root.join(".git").exists();

        // Load gitignore
        let gitignore_patterns = if is_git_repo {
            Self::load_gitignore(&root).await.unwrap_or_default()
        } else {
            Vec::new()
        };

        Ok(WorkspaceContext {
            root,
            project_type,
            context_files,
            skill_directory,
            is_git_repo,
            gitignore_patterns,
        })
    }

    async fn load_context_files(
        root: &Path,
        project_type: ProjectType,
    ) -> Result<Vec<ContextFile>, CoreError> {
        let mut files = Vec::new();

        // Priority order for context files
        let context_specs = vec![
            ("CLAUDE.md", 100),
            ("README.md", 50),
            ("readme.md", 50),
            ("README", 40),
        ];

        for (name, priority) in context_specs {
            let path = root.join(name);
            if path.exists() {
                match fs::read_to_string(&path).await {
                    Ok(content) => {
                        info!("Loaded context file: {}", name);
                        files.push(ContextFile {
                            name: name.to_string(),
                            path: path.clone(),
                            content,
                            priority,
                        });
                    }
                    Err(e) => {
                        warn!("Failed to read {}: {}", name, e);
                    }
                }
            }
        }

        // Project-specific files
        match project_type {
            ProjectType::Rust => {
                Self::try_load(&root, "Cargo.toml", 30, &mut files).await;
            }
            ProjectType::Node => {
                Self::try_load(&root, "package.json", 30, &mut files).await;
            }
            ProjectType::Python => {
                Self::try_load(&root, "pyproject.toml", 30, &mut files).await;
            }
            _ => {}
        }

        // Sort by priority (highest first)
        files.sort_by(|a, b| b.priority.cmp(&a.priority));

        Ok(files)
    }

    async fn try_load(
        root: &Path,
        name: &str,
        priority: u32,
        files: &mut Vec<ContextFile>,
    ) {
        let path = root.join(name);
        if let Ok(content) = fs::read_to_string(&path).await {
            files.push(ContextFile {
                name: name.to_string(),
                path,
                content,
                priority,
            });
        }
    }

    async fn find_skill_directory(root: &Path) -> Option<PathBuf> {
        let candidates = vec![
            root.join(".skilldeck").join("skills"),
            root.join(".claude").join("skills"),
        ];

        for candidate in candidates {
            if candidate.exists() && candidate.is_dir() {
                info!("Found skill directory: {:?}", candidate);
                return Some(candidate);
            }
        }

        debug!("No skill directory found");
        None
    }

    async fn load_gitignore(root: &Path) -> Result<Vec<String>, CoreError> {
        let gitignore_path = root.join(".gitignore");
        if !gitignore_path.exists() {
            return Ok(Vec::new());
        }

        let content = fs::read_to_string(&gitignore_path).await?;

        let patterns: Vec<String> = content
            .lines()
            .map(|line| line.trim())
            .filter(|line| !line.is_empty() && !line.starts_with('#'))
            .map(String::from)
            .collect();

        debug!("Loaded {} gitignore patterns", patterns.len());
        Ok(patterns)
    }

    /// Build context string for AI.
    pub fn build_context_string(context: &WorkspaceContext) -> String {
        let mut parts = Vec::new();

        // Add project type info
        parts.push(format!(
            "Project type: {}\n",
            context.project_type
        ));

        // Add context files
        for file in &context.context_files {
            parts.push(format!(
                "=== {} ===\n{}\n",
                file.name, file.content
            ));
        }

        parts.join("\n")
    }
}

//! Export-related Tauri commands.

use serde::{Deserialize, Serialize};
use std::fs::File;
use std::io::Write;
use std::path::PathBuf;
use tauri::State;
use uuid::Uuid;

use crate::state::AppState;

/// Export format.
#[derive(Debug, Clone, Serialize, Deserialize)]
#[serde(rename_all = "snake_case")]
pub enum ExportFormat {
    Markdown,
    Json,
}

/// Export a conversation.
#[tauri::command]
pub async fn export_conversation(
    state: State<'_, Arc<AppState>>,
    id: Uuid,
    format: ExportFormat,
    path: PathBuf,
) -> Result<(), String> {
    let db = state.registry.db().await.map_err(|e| e.to_string())?;

    // Get conversation
    let conversation = sqlx::query!(
        r#"
        SELECT id, title, created_at
        FROM conversations
        WHERE id = ?
        "#,
        id
    )
    .fetch_one(db.connection())
    .await
    .map_err(|e| e.to_string())?;

    // Get messages
    let messages = sqlx::query!(
        r#"
        SELECT role, content, created_at
        FROM messages
        WHERE conversation_id = ?
        ORDER BY created_at ASC
        "#,
        id
    )
    .fetch_all(db.connection())
    .await
    .map_err(|e| e.to_string())?;

    // Generate content
    let content = match format {
        ExportFormat::Markdown => {
            let mut md = format!(
                "# {}\n\n*Exported on {}*\n\n",
                conversation.title.unwrap_or_else(|| "Conversation".to_string()),
                chrono::Utc::now().format("%Y-%m-%d %H:%M:%S UTC")
            );

            for msg in messages {
                let role = match msg.role.as_str() {
                    "user" => "👤 User",
                    "assistant" => "🤖 Assistant",
                    _ => &msg.role,
                };
                md.push_str(&format!("## {}\n\n{}\n\n", role, msg.content));
            }

            md
        }
        ExportFormat::Json => {
            serde_json::to_string_pretty(&serde_json::json!({
                "id": id,
                "title": conversation.title,
                "created_at": conversation.created_at,
                "messages": messages.iter().map(|m| serde_json::json!({
                    "role": m.role,
                    "content": m.content,
                    "created_at": m.created_at
                })).collect::<Vec<_>>()
            }))
            .map_err(|e| e.to_string())?
        }
    };

    // Write file
    let mut file = File::create(&path).map_err(|e| e.to_string())?;
    file.write_all(content.as_bytes()).map_err(|e| e.to_string())?;

    Ok(())
}

/**
 * Hook for handling agent stream events.
 *
 * Subscribes to agent events and updates streaming text in real-time.
 */

import { useEffect, useCallback } from "react";
import { onAgentEvent } from "@/lib/events";
import { useUIStore } from "@/store/ui";

export function useAgentStream(conversationId: string | null) {
  const appendStreamingText = useUIStore((s) => s.appendStreamingText);
  const clearStreamingText = useUIStore((s) => s.clearStreamingText);

  useEffect(() => {
    if (!conversationId) return;

    let unlisten: (() => void) | null = null;

    const subscribe = async () => {
      unlisten = await onAgentEvent((event) => {
        if (event.conversation_id !== conversationId) return;

        switch (event.type) {
          case "token":
            if (event.delta) {
              appendStreamingText(conversationId, event.delta);
            }
            break;

          case "done":
          case "error":
            clearStreamingText(conversationId);
            break;
        }
      });
    };

    subscribe();

    return () => {
      if (unlisten) unlisten();
    };
  }, [conversationId, appendStreamingText, clearStreamingText]);

  const streamingText = useUIStore(
    (s) => s.streamingText[conversationId ?? ""] ?? "",
  );

  return {
    streamingText,
  };
}

}

export function MessageInput({ conversationId }: MessageInputProps) {
  const [content, setContent] = useState("");
  const [isComposing, setIsComposing] = useState(false);
  const [showCommandPalette, setShowCommandPalette] = useState(false);
  const [showSkillSelector, setShowSkillSelector] = useState(false);
  const [showFileSelector, setShowFileSelector] = useState(false);

  const draft = useUIEphemeralStore((s) => s.drafts[conversationId] || "");
  const setDraft = useUIEphemeralStore((s) => s.setDraft);

  useEffect(() => {
    setContent(draft);
  }, [conversationId]);

  // Save draft on change
  useEffect(() => {
    const timer = setTimeout(() => {
      setDraft(conversationId, content);
    }, 500);
    return () => clearTimeout(timer);
  }, [content, conversationId, setDraft]);

    if (!content.trim() || isComposing) return;

    try {
      await sendMessage(conversationId, content.trim());
      setContent("");

      console.error("Failed to send message:", error);
    }
  };

  const handleKeyDown = (e: React.KeyboardEvent) => {
    if (e.key === "Enter" && !e.shiftKey && !isComposing) {
      e.preventDefault();
      handleSubmit();
    }
  };

    setShowCommandPalette(true);
  }, []);

  const handleAtKey = useCallback(() => {
    setShowSkillSelector(true);
  }, []);

  const handleHashKey = useCallback(() => {
    setShowFileSelector(true);
  }, []);

  return (
    <div className="p-4">
      <div className="relative">
        <Textarea
          ref={textareaRef}
          value={content}
          onChange={(e) => setContent(e.target.value)}

//! Security tests for symlink handling.

use std::fs;
use tempfile::TempDir;

#[tokio::test]
async fn symlink_skill_directory_is_skipped() {
    let dir = TempDir::new().unwrap();

    // Create a skill outside the skill directory
    let external_dir = dir.path().join("external");
    fs::create_dir_all(&external_dir).unwrap();
    let skill_file = external_dir.join("SKILL.md");
    fs::write(&skill_file, "---\nname: external\n---\ncontent").unwrap();

    // Create symlink pointing to external skill
    let skill_dir = dir.path().join("skills");
    fs::create_dir_all(&skill_dir).unwrap();
    let symlink_path = skill_dir.join("symlink-skill");

    #[cfg(unix)]
    {
        std::os::unix::fs::symlink(&external_dir, &symlink_path).unwrap();
    }

    // Scan should skip symlink
    let scanner = skilldeck_core::skills::scanner::SkillScanner::new();
    let skills = scanner.scan(&skill_dir).await.unwrap();

    // The symlinked skill should not be loaded
    assert!(
        !skills.iter().any(|s| s.name == "external"),
        "Symlinked skills should be skipped"
    );

import { render, screen } from "@testing-library/react";
import userEvent from "@testing-library/user-event";
import { describe, it, expect } from "vitest";
import { AppShell } from "@/components/layout/app-shell";

describe("Keyboard Navigation", () => {
  it("all interactive elements are focusable", async () => {
    render(<AppShell />);

    const user = userEvent.setup();
    const focusableElements: HTMLElement[] = [];

    // Tab through all elements
    const maxTabs = 50;
    for (let i = 0; i < maxTabs; i++) {
      await user.tab();
      const focused = document.activeElement as HTMLElement;
      if (focused && focused !== document.body) {
        if (!focusableElements.includes(focused)) {
          focusableElements.push(focused);
        }
      }
    }

    // All buttons, inputs, and links should be focusable
    const buttons = screen.getAllByRole("button");
    const inputs = screen.getAllByRole("textbox");

    expect(focusableElements.length).toBeGreaterThanOrEqual(buttons.length + inputs.length);
  });

  it("command palette opens with Cmd+K", async () => {
    render(<AppShell />);

    const user = userEvent.setup();
    await user.keyboard("{Meta>}k{/Meta}");

    // Command palette should be visible
    expect(screen.getByRole("dialog")).toBeInTheDocument();
  });

  it("can navigate conversation list with arrow keys", async () => {
    render(<AppShell />);

    const user = userEvent.setup();

    // Focus the conversation list
    await user.tab();
    await user.tab();

    // Navigate down with arrow
    await user.keyboard("{ArrowDown}");

    // First conversation should be focused
    const firstConversation = screen.getByRole("button", { name: /conversation/i });
    expect(firstConversation).toHaveFocus();
  });
});
