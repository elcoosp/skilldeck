## Pourquoi SkillDeck ?

SkillDeck est conçu pour les développeurs qui ont besoin d'une IA comprenant leur code – sans compromettre la sécurité ou le contrôle. Vos données restent sur votre machine, vos compétences sont versionnées comme du code, et votre équipe partage des connaissances qui s'accumulent avec le temps.

<CardGrid>
  <Card title="Local en priorité" icon="key">
    Votre code ne quitte jamais votre machine. Les clés API sont stockées dans votre trousseau système, les conversations restent locales.
  </Card>
  <Card title="Compétences comme du code" icon="puzzle">
    Versionnées, partageables sous forme de Gists et composables comme tout autre outil de développement. Installez depuis notre registre ou créez les vôtres.
  </Card>
  <Card title="Flux de travail agentiques" icon="rocket">
    Du simple chat aux orchestrations parallèles complexes. Utilisez des modèles séquentiels, parallèles ou évaluateur‑optimiseur.
  </Card>
  <Card title="Écosystème MCP" icon="link">
    Connectez‑vous à n’importe quel outil via le Model Context Protocol – système de fichiers, GitHub, Slack, bases de données, et plus encore.
  </Card>
</CardGrid>
