Bienvenue sur SkillDeck ! Ce guide vous guidera à travers l'installation de SkillDeck sur votre machine afin que vous puissiez commencer à orchestrer des flux de travail IA localement.

<Nudge type="tip" title="Avant de commencer">
  Assurez‑vous que votre système répond aux [exigences minimales](#exigences-système).
</Nudge>

## Exigences système

SkillDeck fonctionne sur Windows, macOS et Linux. Vous aurez besoin de :

- **Système d'exploitation** :
  - Windows 10 ou ultérieur
  - macOS 12 (Monterey) ou ultérieur
  - Linux (Ubuntu 20.04+, Fedora 38+, ou équivalent)
- **Matériel** :
  - 4 Go de RAM minimum (8 Go recommandés)
  - 500 Mo d'espace disque libre
- **Logiciel** :
  - Pour les modèles IA locaux (facultatif) : [Ollama](https://ollama.com) installé et en cours d'exécution

## Télécharger SkillDeck

1. Visitez la [page de téléchargement officielle](https://skilldeck.dev/download).
2. Choisissez l'installateur pour votre système d'exploitation.
   - Windows : `SkillDeck-Setup-x64.exe`
   - macOS : `SkillDeck-x64.dmg` (Intel) ou `SkillDeck-arm64.dmg` (Apple Silicon)
   - Linux : `skilldeck-x86_64.AppImage`
3. Enregistrez le fichier sur votre ordinateur.

<Checkpoint question="Avez‑vous téléchargé le bon installateur pour votre système d'exploitation ?" />

## Étapes d'installation

### Windows

1. Double‑cliquez sur le fichier `.exe` téléchargé.
2. Si un avertissement de sécurité apparaît, cliquez sur **Plus d'infos** puis **Exécuter quand même**.
3. Suivez les instructions de l'installateur (choisissez l'emplacement d'installation, créez des raccourcis).
4. Cliquez sur **Terminer** pour lancer SkillDeck.

### macOS

1. Ouvrez le fichier `.dmg` téléchargé.
2. Faites glisser l'application **SkillDeck** dans le dossier **Applications**.
3. Ouvrez **Applications** et double‑cliquez sur **SkillDeck**.
   - La première fois que vous l'ouvrez, vous pouvez voir un avertissement indiquant que l'application provient d'un développeur non identifié.
     Allez dans **Réglages système > Confidentialité et sécurité** et cliquez sur **Ouvrir quand même**.
4. Confirmez que vous souhaitez ouvrir l'application.
