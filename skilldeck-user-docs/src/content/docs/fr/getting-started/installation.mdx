Welcome to SkillDeck! This guide will walk you through installing SkillDeck on your machine so you can start orchestrating AI workflows locally.

<Nudge type="tip" title="Before you begin">
  Make sure your system meets the [minimum requirements](#system-requirements).
</Nudge>

## System Requirements

SkillDeck runs on Windows, macOS, and Linux. You'll need:

- **Operating System**:
  - Windows 10 or later
  - macOS 12 (Monterey) or later
  - Linux (Ubuntu 20.04+, Fedora 38+, or equivalent)
- **Hardware**:
  - 4 GB RAM minimum (8 GB recommended)
  - 500 MB free disk space
- **Software**:
  - For local AI models (optional): [Ollama](https://ollama.com) installed and running

## Download SkillDeck

1. Visit the [official downloads page](https://skilldeck.dev/download).
2. Choose the installer for your operating system.
   - Windows: `SkillDeck-Setup-x64.exe`
   - macOS: `SkillDeck-x64.dmg` (Intel) or `SkillDeck-arm64.dmg` (Apple Silicon)
   - Linux: `skilldeck-x86_64.AppImage`
3. Save the file to your computer.

<Checkpoint question="Have you downloaded the correct installer for your OS?" />

## Installation Steps

### Windows

1. Double‑click the downloaded `.exe` file.
2. If a security warning appears, click **More info** then **Run anyway**.
3. Follow the installer prompts (choose installation location, create shortcuts).
4. Click **Finish** to launch SkillDeck.

### macOS

1. Open the downloaded `.dmg` file.
2. Drag the **SkillDeck** app into the **Applications** folder.
3. Open **Applications** and double‑click **SkillDeck**.
   - The first time you open it, you may see a warning that the app is from an unidentified developer.
     Go to **System Settings > Privacy & Security** and click **Open Anyway**.
4. Confirm that you want to open the app.

### Linux

1. Open a terminal in the folder containing the downloaded `.AppImage` file.
2. Make the file executable:
   ```bash
   chmod +x skilldeck-x86_64.AppImage
   ```
3. Run the AppImage:
   ```bash
   ./skilldeck-x86_64.AppImage
   ```
4. (Optional) Move the AppImage to a directory in your `PATH` for easier access.
