SkillDeck is currently in early development. While we work on providing pre‑built installers for all platforms, the recommended way to try the app is to **build it from source**. This guide walks you through the entire process.

<Nudge type="warning" title="Early stage software">
  SkillDeck is under active development. You may encounter bugs or missing features. Your feedback is invaluable – please [report issues](https://github.com/elcoosp/skilldeck/issues) on GitHub.
</Nudge>

## Prerequisites

Before you begin, ensure your system meets the following requirements.

### System Dependencies

#### Windows
- Windows 10 or later
- [Visual Studio Build Tools](https://visualstudio.microsoft.com/downloads/#build-tools-for-visual-studio-2022) with "Desktop development with C++" workload
- [WebView2](https://developer.microsoft.com/en-us/microsoft-edge/webview2/) (usually pre‑installed on Windows 11)

#### macOS
- macOS 12 (Monterey) or later
- Xcode Command Line Tools: `xcode-select --install`

#### Linux
- Ubuntu 20.04+ / Fedora 38+ (or equivalent)
- System packages:
  ```bash
  # Debian/Ubuntu
  sudo apt update
  sudo apt install libwebkit2gtk-4.1-dev build-essential curl wget file libssl-dev libgtk-3-dev libayatana-appindicator3-dev librsvg2-dev

  # Fedora
  sudo dnf groupinstall "C Development Tools and Libraries"
  sudo dnf install webkit2gtk4.1-devel openssl-devel gtk3-devel libappindicator-gtk3-devel librsvg2-devel
  ```

### Software Requirements

| Tool | Version | Installation |
|------|---------|--------------|
| **Node.js** | 20.x or later | [nodejs.org](https://nodejs.org/) or via [nvm](https://github.com/nvm-sh/nvm) |
| **pnpm** | 9 or later | `npm install -g pnpm` or [official instructions](https://pnpm.io/installation) |
| **Rust** | 1.80+ | [rustup.rs](https://rustup.rs/) |
| **Tauri CLI** | 2.x | `cargo install tauri-cli` |

## Step 1: Clone the Repository

Open a terminal and run:

```bash
git clone https://github.com/elcoosp/skilldeck.git
cd skilldeck
```

This downloads the entire source code, including the frontend (`src/`) and the Tauri backend (`src-tauri/`).

## Step 2: Install Frontend Dependencies

The frontend uses pnpm. Install dependencies with:

```bash
pnpm install
```

This will read `package.json` and install all required packages into `node_modules`.

## Step 3: Build the Frontend

Before running the app, you need to build the frontend assets:

```bash
pnpm build
```

This creates an optimized production build in the `dist/` folder, which the Tauri backend will serve.

## Step 4: Build and Run the Tauri App

Now you can build the desktop app:

```bash
cd src-tauri
cargo build --release
```

This compiles the Rust backend and bundles the frontend. The first build may take a few minutes as it downloads dependencies.

After a successful build, you can run the app:

```bash
cargo run --release
```

Alternatively, use the Tauri CLI from the project root:

```bash
pnpm tauri dev
```

This starts the app in development mode with hot‑reload for the frontend.

<Checkpoint question="Did the app launch successfully?" onYes="Great! You're now running SkillDeck from source." onNo="Check the error message – it often points to missing dependencies." />

## Step 5: Post‑Build Configuration

Once the app is running, you can configure it through the Settings panel:

- Add API keys for Claude/OpenAI (see [Set up API keys](/en/how-to/set-up-api-keys))
- Create profiles (see [Configure profiles](/en/how-to/configure-profiles))

## Troubleshooting Build Issues

### Common Errors

| Error | Solution |
|-------|----------|
| `pnpm: command not found` | Install pnpm globally: `npm install -g pnpm` |
| `cargo: command not found` | Install Rust via [rustup](https://rustup.rs/) |
| `Linker error` | Ensure you have the system dependencies installed (see above) |
| `WebView2 not found` (Windows) | Download and install [WebView2 Runtime](https://developer.microsoft.com/en-us/microsoft-edge/webview2/) |
| `Permission denied` (Linux) | You may need to install additional libraries (see distribution‑specific commands) |

### Cleaning Build Artifacts

If you encounter strange errors, try cleaning and rebuilding:

```bash
# Remove frontend dependencies
rm -rf node_modules
# Remove Rust target folder
rm -rf src-tauri/target
# Reinstall and rebuild
pnpm install && pnpm build && cd src-tauri && cargo build --release
```
