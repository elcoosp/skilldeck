La communauté SkillDeck construit une bibliothèque incroyable de compétences réutilisables. Cette page met en avant certaines des contributions les plus utiles, créatives et populaires. Vous voulez voir votre compétence en vedette ? [Partagez‑la avec nous](/fr/community/contribute-skill) !

## Compétences en vedette

### 🐍 Python Code Reviewer
**Auteur :** @pythonista
**Installations :** 1 200+
**Fonction :** Examine le code Python pour la conformité PEP 8, les bugs courants et les améliorations de performances. Inclut des suggestions de correction automatiques.
[Installer depuis le registre](/fr/how-to/install-a-skill) | [Voir sur GitHub](https://github.com/example/python-reviewer)

### 📄 Markdown Formatter
**Auteur :** @docs-whisperer
**Installations :** 850+
**Fonction :** Formate les documents markdown avec une structure de titres cohérente, un alignement des tableaux et une vérification des liens. Parfait pour les rédacteurs de documentation.
[Installer depuis le registre](/fr/how-to/install-a-skill) | [Voir sur GitHub](https://github.com/example/markdown-formatter)

### 🔍 Security Auditor
**Auteur :** @sec-team
**Installations :** 620+
**Fonction :** Analyse les fichiers de dépendances (`package.json`, `requirements.txt`, `Cargo.toml`) pour détecter les vulnérabilités connues et suggère des mises à jour.
[Installer depuis le registre](/fr/how-to/install-a-skill) | [Voir sur GitHub](https://github.com/example/security-auditor)

## Soumettre votre compétence

Vous avez une compétence à mettre en avant ? [Ouvrez une pull request](https://github.com/elcoosp/skilldeck/edit/main/community/showcase.yml) en ajoutant votre compétence à la vitrine. Veuillez inclure :

- Nom de la compétence
- Votre nom/pseudo GitHub
- Brève description (1‑2 phrases)
- Lien vers la compétence (GitHub ou registre)
- Nombre approximatif d'installations (si connu)

Nous examinons les soumissions chaque semaine et mettons en avant les meilleures ici !
