Workspaces are a way to give the agent access to your project's files and structure. When you open a folder as a workspace, SkillDeck scans it and makes its contents available for context injection.

## Why Use Workspaces?

- **Project awareness**: The agent can reference files in your project without manual copy‑paste.
- **Contextual help**: Ask questions about your codebase and get answers based on actual files.
- **Skill development**: Test skills against real‑world code.
