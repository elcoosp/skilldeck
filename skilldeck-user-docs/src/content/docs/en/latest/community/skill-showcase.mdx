## Featured Skills

### 🐍 Python Code Reviewer
**Author:** @pythonista  
**Installs:** 1,200+  
**What it does:** Reviews Python code for PEP 8 compliance, common bugs, and performance improvements. Includes automatic suggestions for fixes.  
[Install from Registry](/en/latest/how-to/install-a-skill) | [View on GitHub](https://github.com/example/python-reviewer)

### 📄 Markdown Formatter
**Author:** @docs-whisperer  
**Installs:** 850+  
**What it does:** Formats markdown documents with consistent heading structure, table alignment, and link checking. Perfect for documentation writers.  
[Install from Registry](/en/latest/how-to/install-a-skill) | [View on GitHub](https://github.com/example/markdown-formatter)

### 🔍 Security Auditor
**Author:** @sec-team  
**Installs:** 620+  
**What it does:** Scans dependency files (`package.json`, `requirements.txt`, `Cargo.toml`) for known vulnerabilities and suggests updates.  
[Install from Registry](/en/latest/how-to/install-a-skill) | [View on GitHub](https://github.com/example/security-auditor)

## Submit Your Skill

Have a skill you'd like to see featured? [Open a pull request](https://github.com/elcoosp/skilldeck/edit/main/community/showcase.yml) adding your skill to the showcase. Please include:

- Skill name
- Your name/GitHub handle
- Short description (1‑2 sentences)
- Link to the skill (GitHub or registry)
- Approximate install count (if known)

We review submissions weekly and feature the best ones here!
