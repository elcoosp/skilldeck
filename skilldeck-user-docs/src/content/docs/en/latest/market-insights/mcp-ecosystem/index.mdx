The **Model Context Protocol (MCP)** is an open standard that connects AI applications to external systems – files, databases, APIs, and more. Think of it like a USB‑C port for AI: a universal way to plug in new capabilities.

## What is MCP?

MCP allows SkillDeck (and other clients) to discover and use tools provided by **MCP servers**. Each server exposes a set of tools (e.g., `read_file`, `search_web`) that the agent can invoke on demand.

## Why MCP Matters for SkillDeck

- **Extensibility**: Add any capability via a server – from filesystem access to GitHub integration.
- **Interoperability**: Use the same server with other MCP‑compatible clients.
- **Security**: Servers run locally, so you control exactly what resources are exposed.
