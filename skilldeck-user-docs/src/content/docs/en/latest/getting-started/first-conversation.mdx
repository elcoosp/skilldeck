Now that SkillDeck is installed, let's have your first conversation with the AI agent.

<Nudge type="tip" title="What you'll learn">
  By the end of this guide, you'll know how to send a message, receive a response, and understand the basics of the agent loop.
</Nudge>

## Launch SkillDeck

Open SkillDeck from your applications menu. You should see the main interface with a chat panel on the left and a sidebar on the right.

## Send Your First Message

1. Click into the message input at the bottom of the chat panel.
2. Type a friendly greeting, for example:
   ```
   Hello! What can you help me with?
   ```
3. Press `Enter` (or click the send button) to send your message.

<Checkpoint question="Did the agent respond to your message?" onYes="Great! You've just experienced the agent loop in action." onNo="Make sure you have an active profile (check Settings → Profiles) and try again." />

## Understanding the Response

The agent will reply with a greeting and list some of its capabilities. By default, SkillDeck has access to built‑in tools and any skills you've installed. The response you see is generated by the model provider configured in your active profile (e.g., Ollama, Claude, or OpenAI).

## What Happens Behind the Scenes

When you send a message:

1. Your message is saved to the local database.
2. The agent loop loads conversation history and any active skills.
3. A request is sent to your chosen model provider.
4. The response streams back to the chat in real time.
5. The final response is saved.

This is the core "agent loop" that powers all interactions.
