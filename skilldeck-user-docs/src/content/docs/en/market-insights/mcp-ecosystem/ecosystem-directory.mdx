The MCP ecosystem has exploded with growth. This directory helps you navigate the landscape of servers, clients, and resources available today.

## 🌐 Official MCP Registry

The **MCP Server Registry** at [registry.modelcontextprotocol.io](https://registry.modelcontextprotocol.io) is the official directory for discovering MCP servers. As of 2026, it hosts **over 200 published servers** ranging from official integrations maintained by companies to community‑contributed implementations [citation:9].
