The command palette is your shortcut to everything in SkillDeck. Press `Cmd+K` (Mac) or `Ctrl+K` (Windows/Linux) to open it, then type to search conversations, skills, actions, and more.

<Nudge type="tip" title="What you'll learn">
  How to open the command palette and use it to navigate and perform common tasks.
</Nudge>

## Opening the Command Palette

- **Keyboard shortcut:** `Cmd+K` (Mac) or `Ctrl+K` (Windows/Linux)
- **Menu:** Click the search icon (magnifying glass) in the top bar, or go to **View → Command Palette**.

You can open it from anywhere in the app.

## What You Can Do

The command palette has several sections:

### 🔍 Search Conversations

Type part of a conversation title to quickly jump to it. The list shows recent conversations matching your query.

### 🧩 Find Skills

Type `@` followed by a skill name to search your installed skills. Selecting one will insert it into your current conversation input (if you're in a conversation) or take you to its detail page.

### ⚡ Actions

Type commands like:

- `new chat` – start a new conversation
- `settings` – open settings
- `workspace` – switch or open a workspace
- `theme` – toggle light/dark mode

### 📄 Open Pages

Type the name of a documentation page or section to jump directly there (if you're viewing docs in the app).

## Examples

| What you want | Type |
|---------------|------|
| Start a new conversation | `new` |
| Open settings | `settings` |
| Switch to "Project X" conversation | `Project X` |
| Find the code‑review skill | `@code-review` |
| Open the Skills tab | `skills` |

## Tips

- The palette learns your most used items and shows them first.
- Use arrow keys to navigate, `Enter` to select.
- Press `Escape` to close without doing anything.
