Workflows let you automate multi‑step tasks by chaining together skills and prompts. In this tutorial, you'll build a **sequential workflow** – steps run in order, one after the other.

<Nudge type="tip" title="What you'll build">
  A workflow that summarizes a document, then translates the summary into French.
</Nudge>

## Prerequisites

- SkillDeck installed and running
- Basic familiarity with the [Skills](/en/latest/getting-started/first-skill) concept
- (Optional) A text file to summarize, or you can use the example below

## Step 1: Open the Workflow Editor

1. In the right sidebar, click the **Workflow** tab (the icon with branching lines).
2. Click the **New Workflow** button (a plus icon or "Create new workflow").
3. A JSON editor will appear – this is where you define your workflow.

## Step 2: Understand the Workflow Definition

A workflow is defined as a JSON object with these main parts:

- `name` – a friendly name for your workflow
- `pattern` – `"sequential"`, `"parallel"`, or `"evaluator_optimizer"`
- `steps` – an array of step objects, each with `id`, `name`, and `prompt` (and optional `skill`)
- `dependencies` – an array of `{ from, to }` objects that define order

For sequential workflows, dependencies are optional because steps run in array order.

## Step 3: Create Your First Sequential Workflow

Copy the following JSON into the editor:

```json
{
  "name": "Summarize and Translate",
  "pattern": "sequential",
  "steps": [
    {
      "id": "step1",
      "name": "Summarize",
      "prompt": "Summarize the following text in 2-3 sentences:\n\n{{text}}"
    },
    {
      "id": "step2",
      "name": "Translate",
      "prompt": "Translate this summary into French:\n\n{{step1.output}}"
    }
  ],
  "dependencies": []
}
```

Notice:
- `{{text}}` is a placeholder – you'll provide the actual text when you run the workflow.
- `{{step1.output}}` refers to the output of the first step. This is how data flows between steps.

<Nudge type="warning" title="JSON validation">
  Make sure your JSON is valid (no trailing commas, quotes properly closed). The editor will highlight errors.
</Nudge>

<Checkpoint question="Did you paste the JSON without errors?" onYes="Great! Let's save it." onNo="Check for missing commas or quotes." />

## Step 4: Save the Workflow

1. Click **Save** (or **Create**).
2. Give it a name like "Summarize and Translate" if prompted (the `name` field already provides it).
3. The workflow should now appear in your list of saved workflows.

## Step 5: Run the Workflow

1. In the Workflow tab, find your new workflow and click **Run**.
2. A dialog will appear asking for input values. Enter the text you want to summarize. For example:

   ```
   Artificial intelligence (AI) is transforming industries by automating tasks, providing insights from data, and enabling new capabilities. In software development, AI tools help with code generation, testing, and documentation. SkillDeck brings these capabilities to your local machine, keeping your code private while leveraging powerful models.
   ```

3. Click **Start** (or **Run**).

## Step 6: Watch the Execution

The workflow will run step by step. You'll see real‑time updates in the UI:

- Step 1 (Summarize) will run first – you'll see its output appear.
- Step 2 (Translate) will then run, using that output as input.

<Checkpoint question="Did both steps complete successfully?" onYes="Awesome! You've built your first sequential workflow." onNo="Check the step prompts – make sure `{{step1.output}}` is spelled correctly." />

## Understanding the Result

The final output is the French translation of the summary. You can use this workflow anytime you need to summarize and translate text.
