SkillDeck skills are built on the **Agent Skills** open standard. This means your skills are portable and can be used with any compatible agent.

For the complete specification, including frontmatter fields, examples, and best practices, please visit:

## [**agentskills.io**](https://agentskills.io/home)

<a href="https://agentskills.io/home" target="_blank" class="external-link">Go to agentskills.io →</a>

---

### Quick Summary

- Skills are defined in a `SKILL.md` file with YAML frontmatter.
- Required fields: `name` (kebab‑case) and `description` (≥20 characters).
- Optional fields: `license`, `version`, `author`, `compatibility`, `allowed_tools`.
- The body contains markdown instructions for the agent.

For the full details, head over to [agentskills.io](https://agentskills.io/home).
