Welcome to SkillDeck! This guide will walk you through installing SkillDeck on your machine.

<Nudge type="tip" cta="Download now" href="https://skilldeck.dev/download">
  **Quick tip:** You can download the latest version from our website.
</Nudge>

## System Requirements

- Windows 10/11, macOS 12+, or Linux (Ubuntu 20.04+)
- 4GB RAM minimum (8GB recommended)
- 500MB free disk space

## Download and Install

1. Visit [skilldeck.dev/download](https://skilldeck.dev/download)
2. Choose your operating system
3. Run the installer and follow the prompts
