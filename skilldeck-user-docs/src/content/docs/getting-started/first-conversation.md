Now that you have SkillDeck installed, let's start your first conversation.

## Open SkillDeck

Launch SkillDeck from your applications menu. You should see the main interface with a chat panel.

## Say Hello

Type "Hello, what can you do?" in the message input and press Enter.

The agent will respond with a greeting and list its capabilities.
